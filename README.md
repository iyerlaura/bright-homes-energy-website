# Bright Homes Energy Website

Website for the Bright Homes Energy Initiative by Athens Area Habitat for Humanity.

## GitHub Pages Setup

This repository is configured to automatically deploy to GitHub Pages using GitHub Actions.

### To Enable GitHub Pages:

1. Go to your repository on GitHub: https://github.com/southernsustainability/bright-homes-energy-website
2. Click on **Settings** (in the repository menu)
3. Scroll down to **Pages** in the left sidebar
4. Under **Source**, select **GitHub Actions**
5. The workflow will automatically deploy your site on the next push

### Your Site URL

Once enabled and deployed, your site will be available at:
**https://southernsustainability.github.io/bright-homes-energy-website/**

### Automatic Deployment

The site will automatically rebuild and deploy whenever you push changes to the `master` branch.
